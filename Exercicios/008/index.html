Principais Formatações

Negrito / Destaque

Neste Utilizei da tag B (não semântica)

Neste utilizei da tag Strong, que é considerada semântica pois ela dá um significado ao uso do negrito, ou seja ela define que está sendo usado negrito para que o texto fique em destaque

Itálico / Ênfase

Neste utilizei da tag i (Não semântica)

Neste utilizei da tag EM(semântica)

Marcação

Pode-se utilizar a tag mark

Tambem posso mudar a cor usando a tag style do css

Letras Grandes e Pequenas

Podemos usar a tag big (Não semântica) e a small (semântica)

Texto deletado

Podemos usar a tag del

Texto inserido

Podemos usar a tag ins (semântica) ou a tag u (Não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3 com a tag sup

Texto subscrito

Para inserir coisas do tipo H2O com a tag sub
